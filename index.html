How Can I Help?
Open Imgur is open source, so if you find any bugs or issues you can either file an issue or submit a pull request with any fixes you have made.

If you would like Open Imgur in another language, you can also contribute by translating the app into the language of your choice. If you are interested in translating Open Imgur into another language and don't know how, feel free to contact me and I can help you.

Feedback is welcomed and encouraged, I hope to make this app as best as possible.
